# 📋 DSA Patterns Cheat Sheet

## 🔁 Sliding Window
- Used for contiguous subarrays/substrings
- Expand right pointer, shrink left when condition breaks
- Examples: Longest Substring, Max Sum Subarray of size K

## 👉 Two Pointer
- Used for sorted arrays, pair finding
- One from start, one from end
- Examples: TwoSum (sorted), Container With Most Water

## 🌊 BFS / DFS
- BFS: Level-order, shortest path (Queue)
- DFS: Depth exploration, backtracking (Stack/Recursion)
- Examples: Binary Tree Level Order, Number of Islands

## 🗺️ Hashing
- O(1) lookup for frequency/existence checks
- Examples: TwoSum, Group Anagrams, Subarray Sum Equals K

## 📊 Binary Search
- Sorted array, O(log n)
- Examples: Search in Rotated Array, Find Peak Element

## 🔗 Fast & Slow Pointers
- Cycle detection in LinkedList
- Examples: Floyd's Cycle Detection

## 🧩 Merge Intervals
- Sort by start time, merge overlapping
- Examples: Merge Intervals, Insert Interval
